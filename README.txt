AIRPORT EAST/WEST TRAINER

Open index.html in a modern web browser.

Features
- Dark-mode classroom interface and dark map tiles.
- ICAO identifier is the largest airport cue, with IATA shown beside it.
- Airport name and location are intentionally smaller to resemble identifier-first flight-plan recognition.
- Select any included airport as the starting/reference airport.
- Quiz airport can be limited to the United States, Mexico, or the Bahamas.
- After an answer is submitted, the map shows both airports, a connecting line, and a north-south longitude guide through the selected reference airport.
- Score, question count, and streak tracking.
- Keyboard shortcuts: E = east, W = west, Enter = submit/next.

Internet connection
The quiz logic and airport data are contained in index.html. An internet connection is required to load Leaflet and the CARTO/OpenStreetMap map tiles.
